# 消费红利策略 · Consumer Dividend Factor Strategy

A 股消费行业红利因子选股策略研究。以股息率（DY）为核心信号，先构建 DY(规模中性) 基础主策略，再引入少量短期反转作为交易过滤，形成最终推荐策略。样本区间为 2013-08 ~ 2022-11，月频调仓。

## 快速概览

| 策略 | 年化收益 | 夏普比率 | 最大回撤 | 平均换手 |
|:---|---:|:---:|:---:|:---:|
| 等权基准 | 14.13% | 0.40 | -39.75% | — |
| 仅股息率 | 14.71% | 0.49 | -27.86% | 9.1% |
| DY(规模中性，基础主策略) | 17.68% | 0.57 | -27.85% | 11.4% |
| **0.7×DY_neut + 0.3×Rev（最终推荐）** | **19.45%** | **0.64** | **-27.33%** | **27.4%** |

> DY(规模中性) 用来验证红利信号本身的有效性和低换手属性；`0.7×DY_neut + 0.3×Rev` 是最终推荐策略。相对等权基准，最终推荐策略年化收益提高约 5.3 个百分点，最大回撤收窄约 12.4 个百分点。收益口径均为价格收益（不含现金分红），基准为等权组合。

## 数据说明

| 字段 | 含义 | 说明 |
|:---|:---|:---|
| `code` | 股票代码 | 261 只深市 A 股消费行业 |
| `close` | 月末收盘价 | 大概率不复权 |
| `divi` | 股息率快照（%） | 百分比形式，每月更新，非分红事件 |
| `cap` | 总市值 | 用于规模中性化和控制变量 |
| `date` | 日期 | 格式 YYYYMMDD，均为月初 |

> **关键发现**：`divi` 是月度股息率快照而非分红事件标识。`close × divi / 100` 还原的 DPS 代理值相邻月近乎不变（92.7% 变化 < 0.01）。DY 因子构建必须走 `dps_proxy → ffill → /close` 路线，直接 ffill divi 会导致 10.5% 样本差异 > 0.1pct。

## 因子库

从 5 个字段可构建 4 类 11 个因子：

| 类别 | 因子 | 多空 t 值 | 结论 |
|:---|:---|---:|:---|
| 红利 | 股息率 DY | 2.67 | 主因子 |
| 红利 | DY(规模中性) | 3.23 | **基础主策略信号** |
| 红利 | 红利稳定性 | 0.90 | 覆盖率下降，排除 |
| 红利 | DPS 增长 | -0.09 | 方向不稳定，排除 |
| 红利 | DY 同比变化 | -0.41 | 方向不稳定，排除 |
| 规模 | ln(市值) | 2.37 | 控制变量 |
| 动量 | 短期反转 | 2.42 | **辅助增强** |
| 动量 | 平滑反转 | 0.71 | 排除 |
| 动量 | 中期动量 | 0.51 | 排除 |
| 动量 | 长期反转 | -0.68 | 排除 |
| 低波 | 低波动 | 0.09 | 辅助观察 |

> EP、BM、换手率、日度低波等因子因数据限制无法构建。

## 关键结论

1. **DY(规模中性) 是基础主策略**。剥离规模噪声后，多空 t 值从 2.67 提升至 3.23，年化收益从 14.71% 升至 17.68%，说明红利信号不是单纯的小盘效应。

2. **简单优于复杂**。红利三因子、精选五因子等复杂合成表现差于纯 DY——弱信号和覆盖率下降共同稀释了主信号。

3. **最终推荐策略为 `0.7×DY_neut + 0.3×Rev`**。该方案以 27.4% 换手实现 19.45% 年化/0.64 夏普，相对等权基准年化收益提高约 5.3 个百分点、最大回撤收窄约 12.4 个百分点。短期反转只作为辅助过滤器，不与红利信号平权竞争。

4. **防御型策略定位**。超额收益集中于 2017-2018 和 2021-2022 的弱市期，趋势牛市中容易跑输。

5. **统计证据谨慎**。随机组合基准 P87.0、Bootstrap 夏普差异 p = 0.12，正向但非强显著——111 个样本月不适合声称发现确定性规律。

## 回测协议

| 参数 | 设定 |
|:---|:---|
| 投资域 | 每月 `close.notna() & cap > 0` 动态筛选 |
| 信号时间 | t 月末信号 → t+1 月收益 |
| 调仓频率 | 月度（季度对照） |
| 选股数量 | Top 15（10/20/30 敏感性；最终推荐策略同样采用 Top15） |
| 权重 | 等权 |
| 费率 | 单边 0.1%（0.002/0.003 敏感性） |
| 基准 | 投资域内等权组合，价格收益，不扣费 |

## 风险与限制

- 收益口径为价格收益，不含现金分红（数据无除息日）→ 对红利策略保守
- 无成交额、换手率、涨跌停等交易微观结构数据 → 无法估计冲击成本
- 无子行业标签 → Top15 可能被动集中在白酒、乳业等子行业
- 全样本仅 111 个有效回测月 → 置信区间较宽，不宜过度外推
- 结论仅适用于消费行业 → 周期/科技/金融板块表现可能不同
